# Yalla-Hagz Website

Yalla-Hagz is the official web frontend for the Yalla-Hagz booking platform. It provides a modern and simple interface that allows users to browse and reserve football, padel, and padbol courts easily.

---

## Live Demo

You can view the live version of the website here:

---

## Overview

Yalla-Hagz Website is a responsive, front-end web project built using HTML, CSS, and JavaScript. It serves as a lightweight client-side interface for the Yalla-Hagz booking service and focuses on usability and clean design.

The project is suitable for learning, prototyping, and showcasing front-end development skills.

---

## Features

### User Interface

- Clean and responsive layout
- Landing page with clear navigation
- Login and signup pages
- Simple and user-friendly booking flow

### Functionality

- Client-side form validation
- Basic authentication logic using JavaScript
- Organized separation of styling and logic files

---

## How It Works

- HTML is used to structure the pages and content
- CSS handles layout, responsiveness, and visual styling
- JavaScript manages:
  - Login and signup logic
  - User interactions
  - Client-side validation and flow control
